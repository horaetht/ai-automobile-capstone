# AI Vehicle Symptom Checker

A simple web-based tool that helps users identify possible car issues based on symptoms and provides repair urgency guidance.

## Features
- Symptom input
- Possible issue classification
- Urgency level
- Recommended next step

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Data: CSV dataset
- Logic: Rule-based symptom matching

## Project Goal
This capstone project aims to build a prototype tool that helps drivers quickly understand possible vehicle problems based on common symptoms.

## Initial Project Structure
- `app/` application code
- `data/` symptom-problem dataset
- `docs/` capstone documents and wireframes
- `tests/` testing files
- `screenshots/` evidence of progress
